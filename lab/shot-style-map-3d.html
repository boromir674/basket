How To Read

The top-left chart shows where the selected team generates shot volume or value. The top-right chart flips the same geometry for what opponents create against that team. The bottom chart puts both on one floor plane.

Encoding

  • Height above zero plane: selected team offense
  • Depth below zero plane: opponent offense versus selected team
  • Color: efficiency delta versus league baseline in the same location
  • Metric toggle changes the column magnitude, not the color baseline logic

Design Choice

This lab uses a smoothed coordinate lattice rather than visible square bins, so the 3D surface reads like a spatial field instead of arbitrary grid blocks cutting through court lines.
